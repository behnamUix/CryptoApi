## 🛠️ Tech Stack & Architecture

This project is built using a modern, reactive Kotlin ecosystem:

*   **Core Framework:** [Ktor Server](https://ktor.io/) (Asynchronous embedded web server powered by Netty & Coroutines).
*   **Dependency Injection:** [Koin Framework v4.0](https://insert-koin.io/) (Lightweight & ultra-fast functional DI engine tailored for Kotlin).
*   **HTTP Client:** [Ktor HTTP Client](https://ktor.io/docs/client.html) (For high-throughput asynchronous fetching from upstream remote APIs).
*   **Containerization:** [Docker](https://www.docker.com/) (Multi-stage lightweight production-ready Docker builds).
*   **Frontend Interface:** Modern, responsive **HTML5/CSS3 Grid Dashboard** featuring an interactive neon glassmorphism layout with native async `fetch()`.

### 🏗️ Clean Architecture Layers

The codebase is strictly modularized into three distinct layers to ensure separation of concerns, high testability, and easy maintainability:

```text
       ┌─────────────────────────────────────────────────────────┐
       │                       Presentation                      │
       │     (Ktor Routes, Dynamic HTML/CSS Dashboards, HTTP)    │
       └────────────────────────────┬────────────────────────────┘
                                    │ (Invokes UseCases via DI)
                                    ▼
       ┌─────────────────────────────────────────────────────────┐
       │                          Domain                         │
       │    (Business Logic, Core Models, UseCases, Interface)   │
       └────────────────────────────┬────────────────────────────┘
                                    │ (Data contract implementation)
                                    ▼
       ┌─────────────────────────────────────────────────────────┐
       │                           Data                          │
       │   (Repositories, Remote Data Sources, Ktor Client API) │
       └─────────────────────────────────────────────────────────┘
